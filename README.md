# 📊 Integrated Retail Analytics Dashboard | SQL & Power BI

## 🚀 Project Overview

This project presents an **Interactive Retail Analytics Dashboard** built using **SQL Server** and **Power BI**. The dashboard provides actionable business insights into sales performance, profitability, customer segments, product performance, and regional analysis to support data-driven decision-making.

---

## 🎯 Key Features

- KPI Tracking
- Sales Trend Analysis
- Product Performance Analysis
- Regional Profitability Analysis
- Customer Segment Analysis
- Interactive Filters and Slicers

---

## 🛠️ Tools & Technologies Used

- SQL Server
- Power BI
- Power Query
- DAX
- Data Modeling

## 💡 Business Insights

- West region generated the highest profit.
- Consumer segment contributed the highest sales.
- Technology category is a key revenue driver.
- Standard Class shipping mode dominates order fulfillment.
